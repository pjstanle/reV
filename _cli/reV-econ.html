reV-econ

reV Economics Command Line Interface

reV-econ [OPTIONS] COMMAND [ARGS]...

Options

--version

Show the version and exit.

-n, --name <name>

reV Economics job name, by default “reV-econ”.

Default

reV-econ

-v, --verbose

Flag to turn on debug logging. Default is not verbose.

direct

Run reV gen directly w/o a config file.

reV-econ direct [OPTIONS] COMMAND [ARGS]...

Options

-sf, --sam_files <sam_files>

Required SAM config files (required) (str, dict, or list).

-cf, --cf_file <cf_file>

Single generation results file (str).

-y, --year <year>

Year of generation results to analyze (if multiple years in cf_file). Default is None (use the only cf_mean dataset in cf_file).

-p, --points <points>

reV project points to analyze (slice, list, or file string). Default is slice(0, 100)

Default

slice(0, 100, None)

-sd, --site_data <site_data>

Site-specific data file for econ calculation. Input should be a filepath that points to a csv. Rows match sites, columns are input keys. Needs a “gid” column. Input as None if no site-specific data.

-spw, --sites_per_worker <sites_per_worker>

Number of sites to run in series on a single worker. Default is the resource column chunk size.

-o, --out_fpath <out_fpath>

Ouput .h5 file path

-lo, --logdir <logdir>

Econ log file directory. Default is ./out/log_econ

Default

./out/log_econ

-or, --output_request <output_request>

Requested output variable name(s). Default is [“lcoe_fcr”].

Default

lcoe_fcr

-ap, --append

Flag to append econ datasets to source cf_file. This has priority over fout and dirout inputs.

-v, --verbose

Flag to turn on debug logging. Default is not verbose.

local

Run econ on local worker(s).

reV-econ direct local [OPTIONS]

Options

-mw, --max_workers <max_workers>

Number of workers. Use 1 for serial, None for all cores.

-to, --timeout <timeout>

Number of seconds to wait for econ parallel run iterations to complete before returning zeros. Default is 1800 seconds.

Default

1800

-pr, --points_range <points_range>

Optional range list to run a subset of sites.

-v, --verbose

Flag to turn on debug logging.

slurm

Run econ on HPC via SLURM job submission.

reV-econ direct slurm [OPTIONS]

Options

-a, --alloc <alloc>

Required SLURM allocation account name.

-no, --nodes <nodes>

Number of SLURM nodes for econ job. Default is 1.

Default

1

-mem, --memory <memory>

SLURM node memory request in GB. Default is None

-wt, --walltime <walltime>

SLURM walltime request in hours. Default is 0.5

Default

0.5

-l, --feature <feature>

Additional flags for SLURM job. Format is “–qos=high” or “–depend=[state:job_id]”. Default is None.

-mod, --module <module>

Module to load

-env, --conda_env <conda_env>

Conda env to activate

-sout, --stdout_path <stdout_path>

Subprocess standard output path. Default is ./out/stdout

Default

./out/stdout

-v, --verbose

Flag to turn on debug logging. Default is not verbose.

from-config

Run reV econ from a config file.

reV-econ from-config [OPTIONS]

Options

-c, --config_file <config_file>

Required reV econ configuration json file.

-v, --verbose

Flag to turn on debug logging. Default is not verbose.

valid-config-keys

Echo the valid Econ config keys

reV-econ valid-config-keys [OPTIONS]
