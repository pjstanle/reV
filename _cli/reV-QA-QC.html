reV-QA-QC

reV QA/QC Command Line Interface

reV-QA-QC [OPTIONS] COMMAND [ARGS]...

Options

--version

Show the version and exit.

-n, --name <name>

reV QA/QC name, by default “reV-QA/QC”.

Default

reV-QA_QC

-v, --verbose

Flag to turn on debug logging. Default is not verbose.

exclusions

Extract and plot reV exclusions mask

reV-QA-QC exclusions [OPTIONS]

Options

-excl, --excl_fpath <excl_fpath>

Required Exclusions file (.h5).

-o, --out_dir <out_dir>

Required Project output directory path.

-sd, --sub_dir <sub_dir>

Required Sub directory to save summary tables and plots too

-exd, --excl_dict <excl_dict>

String representation of a dictionary of exclusions LayerMask arguments {layer: {kwarg: value}} where layer is a dataset in excl_fpath and kwarg can be “inclusion_range”, “exclude_values”, “include_values”, “use_as_weights”, “exclude_nodata”, and/or “weight”.

-afk, --area_filter_kernel <area_filter_kernel>

Contiguous area filter kernel name (“queen”, “rook”).

Default

queen

-ma, --min_area <min_area>

Contiguous area filter minimum area, default is None (No minimum area filter).

-plt, --plot_type <plot_type>

plot_type of plot to create ‘plot’ or ‘plotly’, by default ‘plot’

Default

plotly

Options

plot | plotly

-cmap, --cmap <cmap>

Colormap name, by default ‘viridis’

Default

viridis

-step, --plot_step <plot_step>

Step between points to plot

Default

100

-log, --log_file <log_file>

File to log to, by default None

-v, --verbose

Flag to turn on debug logging.

-t, --terminal

Flag for terminal QA pipeline call. Prints successful status file.

Run reV QA/QC from a config file.

reV-QA-QC from-config [OPTIONS]

Options

-c, --config_file <config_file>

Required reV QA/QC configuration json file.

-v, --verbose

Flag to turn on debug logging. Default is not verbose.

-sd, --sub_dir <sub_dir>

-ds, --dsets <dsets>

-w, --max_workers <max_workers>

Number of workers to use when summarizing 2D datasets, by default None

-plt, --plot_type <plot_type>

plot_type of plot to create ‘plot’ or ‘plotly’, by default ‘plot’
